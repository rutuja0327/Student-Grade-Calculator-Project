# Student Grade Calculator

## 📌 Overview
This Python program calculates student grades based on marks and provides encouraging feedback.

## 🛠️ Features
- Input validation using while loop
- Grade calculation using if-elif-else
- Functions for modular code
- User-friendly messages

## 📝 Grading Logic
- A: 90–100
- B: 80–89
- C: 70–79
- D: 60–69
- F: 0–59
